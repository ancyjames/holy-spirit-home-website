HOW TO VIEW THE WEBSITE:
------------------------

FOR MAC USERS:
1. Double-click "start-website.command"
2. If prompted, click "Open" to allow the script to run
3. Your browser will open automatically
4. To stop: Close the Terminal window

FOR WINDOWS USERS:
1. Double-click "start-website.bat"
2. If Windows asks about security, click "Run anyway"
3. Your browser will open automatically
4. To stop: Close the Command Prompt window

NO ADDITIONAL SOFTWARE REQUIRED!
- Mac: Uses built-in Python
- Windows: Uses built-in PowerShell

TROUBLESHOOTING:
----------------
If the browser doesn't open automatically:
1. Wait for the message "Server started successfully!"
2. Open your browser manually
3. Go to: http://localhost:8000

If Windows blocks the script:
1. Right-click start-website.bat
2. Select "Properties"
3. Check "Unblock" at the bottom
4. Click OK and try again
